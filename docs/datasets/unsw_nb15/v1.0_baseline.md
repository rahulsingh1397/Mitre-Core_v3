## Known Limitations

1. **Worms class sparsity:** Only 130 rows in the full corpus (~7 in a 10K eval subset). Cluster recall for Worms is effectively unmeasurable at this sample size; it is absorbed into larger clusters.
2. **Spectral (raw) high variance:** ARI ranges from 0.16 to 0.48 across seeds (std=0.16). NSL-KDD showed similar behavior — Spectral on raw features is sensitive to initialization seed.
3. **Single checkpoint:** Same `network_v9_v3` checkpoint used for NSL-KDD, with no UNSW-specific fine-tuning. This confirms zero-shot transfer from NSL-KDD to UNSW but likely leaves headroom.
4. **No per-dataset sweep:** Path A decision means we used the NSL-KDD-frozen engine config unchanged. A UNSW-specific sweep might yield ARI >0.564 but was not warranted given the margin.
