## Known Limitations

1. **V3 underperforms baselines** on this dataset (ARI 0.355 vs 0.382 for PCA+HDBSCAN). Documented in `investigation.md`.
2. **Embedding collapse:** The `siem_supcon_v4` checkpoint produces over-smoothed embeddings (mean pairwise cosine similarity = 0.958).
3. **No benign label:** SQTK_SIEM has no explicit benign/normal class; `attack_f1` defaults to 1.0 across all methods.
4. **Small classes:** `permitted` (8 rows), `notified` (42 rows) — cluster recall may be poor.
5. **Single-checkpoint dependence:** Results are specific to `siem_supcon_v4/best.pt`. A generic checkpoint (e.g., `network_v9_v3`) might perform differently.
